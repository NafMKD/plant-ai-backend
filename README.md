### Plant AI Backend
---
This repository serves as the backend component for the [Plant AI](https://github.com/NafMKD/plant-ai) project. For comprehensive project details and documentation, please refer to the [Plant AI](https://github.com/NafMKD/plant-ai) repository.

#### Overview

The Plant AI Backend is a crucial part of the Plant AI project, providing the necessary server-side infrastructure to support various functionalities, including image recognition, data processing, and generative AI.

#### Features

- **Image Recognition**: Utilizes cutting-edge AI algorithms for the accurate identification of plant diseases based on leaf images for more refer [Greenathon-Plant-AI](https://github.com/Rishit-dagli/Greenathon-Plant-AI/tree/main) repository.

- **Generative AI**: uses openai API for generation of responses.

#### Getting Started

To get started with the Plant AI Backend, please refer to the [Plant AI](https://github.com/NafMKD/plant-ai) repository for detailed installation and usage instructions.
1. `git clone https://github.com/NafMKD/plant-ai-backend.git`
5. `cd plant-ai-backend`
2. `python3 -m venv env`
3. `source env/bin/activate`
4. `pip install -r requirements.txt`
7. `python main.py`
